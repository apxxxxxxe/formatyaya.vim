# formatyaya.vim
[formatyaya](https://github.com/apxxxxxxe/formatyaya) for vim

## install
- vim-plug
```
Plug 'apxxxxxxe/formatyaya.vim'
```
- dein.vim
```
[[plugins]]
repo = 'apxxxxxxe/formatyaya.vim'
```

## commands
### `:FmtYaya`
execute formatting

## variables
### `g:formatyaya_option`

commandline options for formatyaya  
default value is `'-s -c 2'`
